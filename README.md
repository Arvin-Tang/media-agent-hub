# Media Agent Hub 🎬

内容创作自动化仓库。灵感从飞书群「点子王」诞生，经过 Agent 加工，变成 AI 主题的文章、视频、漫画作品。

## 目录结构

```
media-agent-hub/
│
├── insights/                    ← 灵感挖掘：每日洞察 + AI选题
│   └── YYYY-MM-DD/
│       ├── 群聊洞察-AI-YYYY-MM-DD.md   # 当天AI话题趋势分析
│       └── AI选题-YYYY-MM-DD.md        # 精选AI选题包
│
└── works/                        ← 创作作品（每作品独立目录）
    │
    ├── 文章-YYYY-MM-DD-标题/          ← 文章作品
    │   ├── 草稿.md                     # 文字初稿
    │   └── ...                         # 后续可加：终稿.md / 配图/
    │
    ├── 视频-YYYY-MM-DD-标题/          ← 视频作品
    │   ├── 手稿/                       # 脚本/逐字稿/分镜
    │   ├── 素材/                       # BGM/封面/画面素材
    │   └── 成品/                       # 渲染完成的视频文件
    │
    └── 漫画-YYYY-MM-DD-标题/          ← 漫画作品
        ├── 手稿/                       # 分镜脚本/角色设定/对白
        ├── 素材/                       # 角色设定图/背景图/元素
        └── 成品/                       # 最终组图/条漫
```

## 命名规范

| 前缀 | 含义 | 示例 |
|------|------|------|
| `文章-` | 文字作品 | `文章-2026-06-11-AI技术名词一本通/` |
| `视频-` | 视频作品 | `视频-2026-06-11-AI技术名词一本通/` |
| `漫画-` | 漫画作品 | `漫画-2026-06-11-AI打工人的一天/` |

每个作品文件夹下按 **手稿→素材→成吕** 分层，清晰可追溯。

## 工作流

```
群聊消息  →  AI筛选 → 洞察分析 → AI选题
                                    ↓
                            分发创作（文章/视频/漫画）
                                    ↓
                产出到 works/ 对应目录 → git push
```
