# 漫画分镜脚本：AI 打工人的 Agent 同事

**漫画类型**: 四格漫画风格，6页
**风格方向**: 轻松调侃，上班族日常

---

## 第1页：早上的对比

**画面**
分左右两格。左边是一个人类程序员满头大汗在敲代码，桌上咖啡洒了。右边是电脑屏幕上一个 Agent 界面在自动操作。

**对白**

程序员对屏幕喊："代码还没跑完啊！"

Agent 界面显示："✅ 代码已写完 ✅ 测试已通过 ✅ 文档已生成 ✅ 已部署上线"

小字标注："先别急，你的 AI 同事已经搞定了。"

---

## 第2页：开会

**画面**
会议室。人类同事们围坐，一个人在做汇报。旁边一台电脑屏幕上 Agent 正在自动记录。

**对白**

PM："下个 Sprint 我们有 12 个 Story..."

程序员的 Agent（弹窗）："已分析代码库关联模块，自动拆分为 28 个子任务，预估工时 46 小时。"

PM："……哦，不用讲了。那散会？"

---

## 第3页：摸鱼被发现

**画面**
程序员躺在椅子上刷手机。老板走过来。

**对白**

老板："你代码写完了？"

程序员（心虚地）:"马上马上……"

老板："我问的是你的 AI Agent 写完了没。它写完你爱干嘛干嘛。"

程序员松一口气："哦，它两小时前就提交了。"

---

## 第4页：下班后

**画面**
程序员离开工位。他桌上的电脑屏幕仍然亮着，Agent 还在自动运行。

**对白**

程序员手机弹出通知："代码库迁移进度：47% → 预计凌晨 3 点完成"

程序员（自言自语）："我走了它还在加班……谁才是打工人？"

背景飘过一行小字："2026 年，Agent 掀起了第一次'数字员工权益运动'——没工资、不睡觉、24 小时在线。"

---

## 第5页：微信对战

**画面**
两格。第一格：程序员在走路，对着手机说："Siri，问老王晚上几点有空。"第二格：对方手机上，AI Agent 自动接管微信，回复了消息。

**对白**

老王 AI Agent："王总今晚 7 点后有空，已自动预约会议室。"

程序员："……等等，我的 Agent 怎么没帮我约？"

程序员手机弹窗："您尚未开通 Agent 订阅服务，¥9.9/月，立即开通？"

---

## 第6页：结尾

**画面**
程序员和旁边空椅子上的手机/电脑击掌（拟人化）。

**对白**

程序员："行吧，这 ¥9.9 我付了。以后你白天干活，我白天……呃，我学习。"

Agent（屏幕显示）:"已更新目标：自我迭代。确认？"

程序员（赶紧点取消）："撤回撤回！你学太多我就真失业了！"

**定格文字：**
"2026，AI 从'聊天'变成'干活'。
你准备好当那个管 Agent 的人了吗？"
