# AI Agent 三篇价值选题创作规划

## 选题组合

本组优先创作三个高价值主题：

1. Loop Engineering
2. AI 编程 Agent 时代
3. HermesAgent

它们不是孤立选题，而是一条完整内容线：

- Loop Engineering：方法论母题，解释“为什么要从 Prompt 转向 Loop”
- AI 编程 Agent 时代：开发者场景，解释 Agent Mode 如何改变编程
- HermesAgent：个人 Agent 场景，解释长期记忆和能力沉淀如何让 AI 越用越懂你

## 推荐发布顺序

### 第一篇：Loop Engineering

- 角色：系列开篇 / 方法论母篇
- 主标题方向：《别再只会写 Prompt：真正厉害的人已经在设计 Loop》
- 目标：建立“Prompt → Loop”的核心认知
- 可复用资产：已有漫画成品，可直接做图文联动

### 第二篇：AI 编程 Agent 时代

- 角色：开发者落地篇
- 主标题方向：《AI 编程进入 Agent 时代：程序员的工作真的变了》
- 目标：把 Loop 方法论落到代码生产场景
- 适合衍生：3 分钟视频、开发者向短视频、技术公众号文章

### 第三篇：HermesAgent

- 角色：个人 Agent / 长期记忆篇
- 主标题方向：《别再只会用 Chatbot，下一代 AI 是会成长的 Agent》
- 目标：从工具使用讲到“数字同事养成”
- 适合衍生：漫画、个人助理场景视频、AI 产品趋势文

## 统一叙事

一句话主线：

> AI Native 的核心，不是多会写 Prompt，而是能不能把任务变成可持续运转的系统。

三篇之间的关系：

```text
Loop Engineering
  ↓ 提供方法论
AI 编程 Agent 时代
  ↓ 展示开发者场景
HermesAgent
  ↓ 展示个人长期 Agent 场景
```

## 建议生产节奏

- 重写三篇文章，统一风格和事实核验标准
- 每篇都加入“发布前事实核验”脚注或谨慎表述
- 输出：公众号版 + 即刻短版 + 小红书轮播文案

### Batch 2：视频

优先做：Loop Engineering 视频完整版。

原因：

- 主题最底层，可复用为系列开篇
- 已有漫画成品，可提取视觉符号
- 可衔接刚完成的 PromptSkillHookLoop 视频

之后再做：AI 编程 Agent 时代视频。

HermesAgent 可以先做文章和漫画，等事实核验更充分再做视频。

### Batch 3：漫画/轮播

- Loop Engineering：已有成品，补发布文案
- AI 编程 Agent 时代：新做漫画/轮播
- HermesAgent：新做漫画，突出“AI 养成/数字同事”
