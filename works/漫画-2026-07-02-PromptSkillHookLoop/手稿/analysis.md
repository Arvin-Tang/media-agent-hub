## Value Proposition

读者看完后要获得一个清晰心智模型：

> Prompt 解决一次对话，Skill 沉淀一类能力，Hook 捕捉一个时刻，Loop 让 AI 进入工作流并持续运转。

这不是单纯概念科普，而是帮助读者判断自己当前处在哪一层，并知道下一步如何升级。

## Core Themes

1. **从对话到系统**：AI 的价值不止在回答，而在进入流程。
2. **重复劳动的结构化**：高频 Prompt 应该沉淀成 Skill。
3. **事件触发**：Hook 让 AI 不必等人开口。
4. **循环运转**：多个 Hook + Skill 串起来，形成 Loop。
5. **工程化记忆**：聊天记录不是工程系统，文件系统才是。
