成品/

## 功能亮点

✅ **自动播放模式** — 点击「▶ 自动播放」按钮，自动翻页 + 语音朗读
✅ **Web Speech API TTS** — 使用浏览器内置语音合成，无需安装任何软件
✅ **字幕条** — 底部的黑色半透明字幕条随每步更新
✅ **静音开关** — 可随时关闭语音仅看画面
✅ **键盘控制** — → / 空格推进，← / Backspace 后退
✅ **进度点** — 已看/当前/未看的圆点指示器
✅ **纯前端** — 无需服务器，双击 HTML 文件即可在浏览器打开

## 如何使用

1. 双击 `00-目录.html` 在浏览器中打开
2. 点击任意章节进入
3. 点击舞台或按 → 手动推进
4. 或点击「▶ 自动播放」让章节自动播完
5. 浏览器会在首次点击时请求语音权限，请允许

## 技术说明

- TTS 使用 `SpeechSynthesisUtterance`，自动选择系统中文语音
- 各章节自动播放的等待时间：Step 0 约 6s, Step 1 约 5s, Step 2+ 约 7s
- 第6章为逐个揭晓式，每次揭晓一个卡片，等待约 3.5s
- Chrome/Safari/Edge 均支持，建议使用 Chrome 获得最佳语音效果
