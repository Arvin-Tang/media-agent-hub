# HermesAgent 创作计划

## 定位

- 主题价值：解释“会成长的个人 Agent”为什么重要
- 目标读者：AI 工具重度用户、独立开发者、自动化爱好者、想搭私人助理的人
- 核心标题方向：别再只会用一次性 Chatbot，下一代 AI 是会养成的 Agent
- 核心钩子：普通 AI 用完就忘，HermesAgent 的卖点是记忆、技能和自进化

## 主表达

HermesAgent 值得讲，不是因为它又是一个 Agent 框架，而是因为它把“长期使用”本身变成了产品能力：记忆沉淀、技能复用、任务自动化、子智能体协作，最终形成一个越用越懂你的私人工作系统。

## 推荐产物

### 1. 公众号长文

- 方向：科普 + 趋势判断
- 标题候选：
  1. 《越用越聪明的 AI 来了：HermesAgent 到底是什么？》
  2. 《别再只会用 Chatbot，2026 年真正值得关注的是“会成长的 Agent”》
  3. 《HermesAgent 火了：AI 从工具，变成了可以培养的数字同事》
- 结构：
  1. 开场：为什么普通 Chatbot 用完就忘
  2. HermesAgent 一句话定义
  3. 三层成长机制：记忆 / Skill / 训练数据
  4. 和 OpenClaw / 普通 Agent 的区别
  5. 适合谁用
  6. 风险与现实边界
  7. 总结：未来不是找一个 AI，而是养一个工作系统

### 2. 视频脚本

- 形式：3 分钟横屏解释视频
- 节奏：
  1. 0-15s：普通 AI 为什么越用越累
  2. 15-45s：HermesAgent 是什么
  3. 45-90s：记忆层
  4. 90-135s：技能层
  5. 135-165s：自动任务 / 子智能体
  6. 165-180s：结论与行动建议

### 3. 漫画/图文

- 角色设定：
  - 小智：每次都重新教 AI 的打工人
  - Hermes：会记事、会整理技能的“数字管家”
  - 普通 Chatbot：健忘但很热情
- 分镜方向：
  1. 小智崩溃：第 18 次教 AI 写日报格式
  2. 普通 Chatbot：对不起，我忘了
  3. Hermes 登场：我已经把流程存成 Skill 了
  4. 记忆柜 / 技能库 / 定时器三件套
  5. 小智：这不是工具，是同事
  6. 金句：AI 的下一站，不是更会聊天，而是更会和你一起成长

## 发布前事实核验

草稿里有些表述需要发布前核验，不要直接当事实发：

- HermesAgent 是否真实存在及 GitHub 地址
- “2026 年 2 月发布”“两个月 4.7 万 Star”“Nous Research”是否准确
- OpenClaw Star 数、开发者信息、模型生态对比是否准确
- “自动提炼 Skill”“训练数据层”是否属于 HermesAgent 官方能力还是文章概念化表达

## 优先级

高。这个主题适合做成“个人 Agent/数字同事”方向的入口内容，可与 OpenClaw、Skill、Loop Engineering 形成系列。
